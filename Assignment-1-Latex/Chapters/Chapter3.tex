\chapter{Research Objectives}
\begin{enumerate}
    \item Create a hybrid retrieval system that to dymacially weigh and route queryes between vector and graph-based retireval with modifying the underlying Large Language Model.
    \item Develop and evaluate rule-based, llm-based, and model-based classifiers to weigh incoming queries between vector and graph retrieval.
    \item Evaluate if a weighted blending strategy can outperform fixed retrieval strategies.
    \item Analyse which query characteristics should influence the retrieval weighting and if there is a threshold to completely remove a strategy.
\end{enumerate}
